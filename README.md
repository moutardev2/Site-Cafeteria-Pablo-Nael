# 🍽️ Cafeteria Django App

> Application web de gestion d’une cafétéria scolaire développée avec Django  
> 🎯 Objectif : gérer les étudiants, les produits et les transactions via une interface moderne.

---

## 🚀 Aperçu du projet

Ce projet a été réalisé dans le cadre d’un TP pour découvrir le framework Django et les bases du développement web.

L’application permet de :

- 👨‍🎓 Gérer les étudiants et leur crédit  
- 🛒 Gérer les produits disponibles (prix, stock, disponibilité)  
- 💳 Enregistrer les transactions  
- 📊 Visualiser les données via une interface moderne avec Bootstrap  

---

## 🧠 Objectifs pédagogiques

Ce projet nous a permis de comprendre :

- L’architecture **MVT (Model - View - Template)**  
- La création de modèles et la gestion d’une base de données  
- Le fonctionnement des vues et des templates  
- L’utilisation de Bootstrap pour le design  
- La gestion d’un environnement Python (`venv`)  
- L’utilisation de Git et GitHub  
- Les bases réseau : localhost, IP, LAN  
- Le CRUD et les requêtes HTTP (GET / POST)
